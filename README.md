# Ulysses: A Personalized Cognitive AI Assistant

> "Artificial Intelligence is more than just code; it is an endeavor to replicate the most complex system in the universe: the human brain."

## Project Philosophy
Ulysses is born from the intersection of **Formal Logic, Philosophy of Mind, and Computer Science**. As a current Philosophy student at Sakarya University, I designed this assistant to move beyond simple automation. My goal is to explore how neuromorphic principles and cognitive psychology can be integrated into software architecture to create a more intuitive user-machine interaction.

## Modular Architecture
The project is structured to mirror specific cognitive functions, allowing for independent development of each "brain" region:
- **`wakeword/`**: The auditory cortex; handling initial stimulus detection.
- **`stt/` (Speech-to-Text)**: Linguistic processing; converting acoustic signals into symbolic thought.
- **`llm.py` & `prompts/`**: The prefrontal cortex; handling formal logic, reasoning, and decision-making.
- **`memory/`**: Cognitive storage; designed to handle both short-term context and long-term personality traits.
- **`skills/`**: The motor cortex; executing specialized tasks and external interactions.

## Tech Stack
- **Language**: Python 3.x
- **Core Intelligence**: Large Language Models (LLM) integration via `llm.py`
- **Voice Processing**: Real-time STT and Wake Word detection
- **Logic Foundation**: Built on principles of formal logic and self-directed research
